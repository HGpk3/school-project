# Контекст проекта

## Задача

Сделать качественный демонстрационный прототип игры «Своя игра» на тему географии России для школьного показа.

## Основные ограничения

- игра должна запускаться с флешки;
- интернет на месте показа может отсутствовать;
- желательно обойтись без установки программ и без сложной настройки;
- игра должна быть понятной ведущему прямо во время урока или выступления;
- нужно поддержать 3 команды.

## Что считается успехом

- ведущий открывает проект и сразу может начать игру;
- счёт ведётся удобно и прозрачно;
- есть финальный раунд и итоговая таблица;
- оформление выглядит аккуратно и цельно;
- проект можно показать на обычном школьном компьютере.

## Принятые решения

- интерфейс делается в зелёной цветовой гамме по запросу;
- игра ориентирована на живого ведущего, который управляет ходом партии;
- логика остаётся близкой к шоу, но там, где телевизионные нюансы мешают удобству, приоритет отдаётся понятному школьному использованию;
- состояние сохраняется локально в браузере, чтобы случайная перезагрузка страницы не ломала игру.
