# Своя игра: География России

Офлайн-прототип школьной игры в формате «Своя игра» на тему географии России.

## Что реализовано

- 2 полноценных раунда с игровым табло.
- 3 команды со счётом, переименованием и ручной корректировкой очков.
- Основные механики шоу:
  - обычные вопросы;
  - `Аукцион`;
  - `Кот в мешке`;
  - финальный раунд с отбором темы, ставками и итоговым подсчётом.
- Журнал событий по ходу игры.
- Скачивание текстового протокола после игры или прямо во время партии.
- Автосохранение состояния в браузере через `localStorage`.
- Полностью офлайн-запуск без интернета и без установки программ.

## Как запустить

1. Скопируй всю папку проекта на флешку.
2. На школьном компьютере открой папку.
3. Дважды щёлкни по файлу `index.html`.
4. Игра откроется в браузере.

## Что важно для демонстрации

- Интернет не нужен.
- Node.js, npm и другие программы не нужны.
- Лучше копировать именно **всю папку**, а не один файл, чтобы стили и логика работали корректно.

## Структура проекта

- `index.html` — оболочка приложения.
- `styles.css` — зелёная тема и адаптивный интерфейс.
- `app.js` — вопросы, механики игры, логика счёта, финал, экспорт протокола.
- `docs/STACK_AND_PLAN.md` — выбранный стек и план реализации.
- `docs/PROJECT_CONTEXT.md` — контекст проекта и ограничения.
- `WORKLOG.md` — журнал выполнения работы.

## Как вести игру

1. Введи названия команд в левой панели.
2. При необходимости назначь команду, у которой право выбора вопроса.
3. Нажимай на стоимость вопроса на табло.
4. В открывшейся карточке отмечай:
   - кто ответил верно;
   - кто ошибся;
   - никто не ответил.
5. После второго раунда перейди в финал, выбери тему через механику удаления тем, задай ставки и отметь ответы.
6. В конце скачай протокол кнопкой `Скачать протокол`.

## Примечание

Прототип ориентирован на школьную очную игру с ведущим. Поэтому спорные телевизионные моменты намеренно сделаны удобными для ручного управления: ведущий может корректировать очки и назначать право хода вручную.
